Security

Honest about what's verified and what's in progress.

SmashOne is a 2026 product. We treat security as a build-not-claim discipline. Here's exactly what's in place today, what's underway, and what we're working toward.

Verified today

What's already in place.

Stripe-verified payments

All card data is processed by Stripe (PCI DSS Level 1, the highest tier). We never see, store, or transmit card numbers. Stripe's tokenization replaces sensitive data with non-sensitive equivalents.

In production

Encryption everywhere

TLS 1.3 for all traffic in transit. AES-256 for all data at rest in our database, backups, and object storage. Encryption keys rotate quarterly.

In production

Florida C-Corp accountable entity

SMASHONE CORPORATION is a registered Florida C-Corporation (registration P26000023598). A real US business, with US tax filings and US legal exposure — not a shell.

Active

US data residency

Database (PostgreSQL), application servers, and encrypted backups all hosted in US-east regions (DigitalOcean NYC1/NYC3, AWS S3 us-east-2). No transcontinental replication.

In production

In progress

What we're actively building.

SOC 2 Type II audit

Independent compliance audit covering security, availability, processing integrity, confidentiality, and privacy. Engagement signed; observation period 2026-Q2 through 2026-Q4. Report expected 2027-Q1.
