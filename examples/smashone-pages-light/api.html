What we're building

Scope of v1.

REST API

Create posts, manage scheduling, list connected platforms, fetch analytics. OAuth 2.0 + scoped API keys. Idempotency keys on every mutation.

Webhooks

Real-time delivery for post.published, dm.received, ai.reply.sent, subscription.changed. Signed payloads. At-least-once delivery, retries with exponential backoff.

Official SDKs

JavaScript/TypeScript, Python, Ruby, and PHP at launch. Auto-generated from the OpenAPI 3.1 spec, manually polished for ergonomics.

OpenAPI 3.1 spec

Full machine-readable spec published with v1. Generate your own client in any language. Versioned at the path level (/v1/, /v2/).
