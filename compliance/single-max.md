# Single‑MAX iron rules (публичный сайт)

## Запрещено упоминать на сайте (user-facing)

- Telegram / Телеграм / t.me
- ВКонтакте / VK / vk.com
- Одноклассники / ОК / ok.ru
- Дзен / dzen.ru
- Automator / «программа автоматизаторов»

Даже в формате «раньше было…» — публичный контент должен быть чистым single‑MAX.

## Запрещённые цены/планы

- «СММ ПРО», «СММ СТАНДАРТ»
- «3990 ₽», «5990 ₽»
- «под ключ 24990» / «24 990»

## Разрешено (канон)

- MAX (латиницей)
- СММ АЙБРО
- 4 990 ₽/мес
- Триал 5 дней
- ОРД: 30 рекламных постов/мес с маркировкой и `erid`; инфо‑посты — без лимита

## Self-check команды

```bash
# Single‑MAX violations
grep -rn "Telegram\|Телеграм\|t\.me\|ВКонтакте\|\bVK\b\|vk\.com\|Одноклассники\|\bОК\b\|ok\.ru\|Дзен\|dzen\.ru\|Automator\|автоматизатор" web/src/pages/

# Тариф/планы violations
grep -rn "СММ ПРО\|СММ СТАНДАРТ\|3990\|24990\|под ключ" web/src/pages/
```
