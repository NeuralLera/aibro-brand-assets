# `sonya/` — С.О.Н.Я (ИИ-ассистент бренда)

> **🔴 ВИЗУАЛ ОБНОВЛЁН 04.06.2026 (решение Валерия): С.О.Н.Я — зелёная девушка-ассистент с гарнитурой.** Старый чёрный «робот-астронавт» (17.05) ОТМЕНЁН и удалён. Не воссоздавать.

## Что это

**С.О.Н.Я** — наш собственный ИИ-ассистент бренда СММ АЙБРО, который:
- ведёт наш MAX-канал (Publisher + Manual Composer),
- отвечает на сообщения в нашем чате сообщества (Group Mode + Responder),
- общается с лидами в личке `max.ru/id519041916737_1_bot`.

Это **наш голос** — НЕ продукт, НЕ центральный логотип. Не путать с СММ АЙБРО БОТ (см. `../aibro-bot/`) и НЕ с центральным логотипом (см. `../main-logo/`).

## Расшифровка имени

**С.О.Н.Я** — **С**ообщество, **О**тветы, **Н**овости, **Я**ркие моменты — 4 runtime-функции (`src/services/aibro_channel/`): Group Mode (`group_mode.py`) · Responder (`responder.py`) · Publisher (`publisher.py`) · Manual Composer (`manual_composer.py`). Согласовано Валерием 17.05.2026.

## Визуальный канон (04.06.2026)

**Зелёная девушка-ассистент с call-центр-гарнитурой.** Изумрудные волосы (gradient emerald), дружелюбная улыбка, румянец, emerald-accent (звёздочка-блик у виска). Тёплый flat-cartoon, светлый mint-halo фон. Образ «дружелюбный эксперт-помощник», НЕ робот.
