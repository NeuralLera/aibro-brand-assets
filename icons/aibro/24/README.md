# Aibro Icon Pack (24px)

Source of truth for reusable stroke icons (public site + help + documents + partner).

- Grid: 24x24
- Style: stroke-based, rounded caps/joins
- Intended usage: UI nav, cards, badges

Files:

- `*.svg` - individual icons
- `sprite.svg` - combined sprite
- `manifest.json` - icon list
- `max/` - MAX icon variants from the pack

Update policy:

- Keep names stable.
- Add new icons by appending files + updating `sprite.svg` and `manifest.json`.
